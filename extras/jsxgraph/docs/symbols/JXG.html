<static> {Object} JXG.paletteWong
Bang Wong color palette, optimized for various type of color blindness. It contains values for
  • 'black'
  • 'orange'
  • 'skyblue'
  • 'bluishgreen'
  • 'yellow'
  • 'darkblue'
  • 'vermillion'
  • 'reddishpurple'
As substitutes for standard colors, it contains the following aliases:
  • black (= #000000)
  • blue (= darkblue)
  • green (= bluishgreen)
  • purple (= reddishpurple)
  • red (= vermillion)
  • white (= #ffffff)
See Bang Wong: "Points of view: Color blindness" and https://davidmathlogic.com/colorblind/.
Defined in: color.js.
