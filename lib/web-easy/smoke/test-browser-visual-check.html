web-easy visual check page

Manual visual sanity page (not part of dashboard PASS/FAIL automation).
  • Check fieldset/group border and spacing (the frame with legend "Visual Check").
  • Check action chips look clickable and have horizontal separation.
  • Check table border/cell spacing and separation from status text.
  • Check tabs have distinct selected/unselected visuals and keyboard focus ring.
  • Check input/checkbox/choice/slider/progress alignment and spacing.
  • Expected widths/alignment:
    group: full-width, left
    input: full-width, left
    checkbox/choice/slider/progress: content-width, left
    table: content-width, left
    tab-panel: full-width, left
    menu-bar: row left; action chips content-width
